# IImage interface

Represents an image.

---
## Methods
### `getImageUrl()` → `String`

Image Url.

#### Return

**Type**

String

**Description**

The image Url.

### `getType()` → `String`

The type of image this is. For example, cover, back, main, thumbnail, etc.

#### Return

**Type**

String

**Description**

The type of image it is.

### `setImageUrl(String imageUrl)` → `void`

Sets the Image Url.

#### Parameters
|Param|Description|
|-----|-----------|
|`imageUrl` |  The Image Url to set. |

### `setType(String type)` → `void`

Sets the image type.

#### Parameters
|Param|Description|
|-----|-----------|
|`type` |  The image type to set. |
